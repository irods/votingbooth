iRODS Consortium Voting Booth
=============================

Provides an OAuth wrapped, minimal interface into a GitHub-backed voting mechanism for the iRODS Consortium.

There is no local knowledge about users, features, votes, etc.  All data is public and timestamped.

Requirements
============

A local copy of https://github.com/csphere/GithubApiWrapper-PHP
